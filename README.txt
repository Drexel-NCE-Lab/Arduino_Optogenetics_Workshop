### Optogenetic Stimulation Control

Author: Liudi Luo 
Lab: Catherine von Reyn Lab
Date: August 3rd 2020
Operation platform: Arduino IDE

Description: 
This is the Arduino resourse which was developed to synchronize light activation for optogenetics experiments with mobile phone video 
capture to record the behavior of transgenic Drosophila melanogaster.






### Notes:
Remember to install Low Power Library (Low-Power-master.zip) before running the code. This library is designed by Rocket Scream Electronics.
